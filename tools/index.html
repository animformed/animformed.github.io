Modular Rigging Tools for Maya -  v1.2  -  Now fully open source and updated, it's an approach to a non-destructive, modular rigging workflow that allows you to set-up any type of character. You can create bipeds, quadrupeds, or any arbitrary type, mechanical rigs and so on. You can save and re-use any type of rig and skeletal configuration for a character. You can also apply multiple control rigs to a joint hierarchy in a character. For example, a joint hierarchy for a spine can be applied with IK/FK controls and reverse spine FK controls at the same time. In addition to built-in control rigs that can be applied for Modular Rigging Tools, a TD can also write a new or extend control type for a joint hierarchy using a python template.

Constrain transform to shape -  v1.1 - A script which allows you to constrain multiple transforms to a shape node, such as, on a polygon mesh, a NURBS surface or a curve. You can maintain the current position(s) of the transform(s), or you can snap them to the shape. Any deformation on the shape node will drive the transform(s) as well. For more details, see here
